# XYZ Store Sales Analysis Dashboard (Excel)
## Project Overview

This project showcases an interactive Excel dashboard created to analyze annual sales performance of XYZ Store for the year 2025.
The dashboard provides insights into sales trends, customer demographics, order status, state-wise performance, and sales channels, helping businesses make data-driven decisions.

## Key Features

Interactive Excel dashboard

Monthly Orders vs Sales analysis

Men vs Women sales comparison

Order status tracking (Delivered, Cancelled, Returned, Refunded)

Top 5 states by total sales

Age group vs gender order analysis

Channel-wise sales distribution

Slicers for Month, Category, and Sales Channels

Tools & Technologies

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

Data Cleaning & Formatting

# Dataset

Sales data of XYZ Store (2025)

Fields include:

Order Date

Sales Amount

Order ID

Customer Gender & Age Group

Product Category

State

Sales Channel

Order Status

Dataset used for learning and analytical purposes only.
